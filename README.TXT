A Commandline Tool For Getting Info For WINDOWS PCs.

How To Use?
 First,copy the files to C:\Windows\System32 or install me.
 Then,you can use the following commands:
 1. > SGET
 2. > SGET-CPUNAME
 3. > SGET-MAXMEM
 4. > SGET-WINVER
